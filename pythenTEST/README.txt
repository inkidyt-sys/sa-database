DSA Helper (石器時代輔助程式 v4.x)
這是一個專為「石器時代」(DSA 新端 / sadsa.exe) 開發的 Python 外部輔助工具。
本程式採用多執行緒架構讀取遊戲記憶體，提供實時的角色與寵物數據監控，並整合了常用的遊戲功能修改（如加速、穿牆等）。

核心功能 (Features)

1. 多客戶端管理
支援同時綁定與監控最多 6 個遊戲視窗。
自動偵測並列出可綁定的遊戲進程 (PID)。
支援個別視窗的隱藏/顯示、縮小/還原操作（透過右鍵點擊）。

2. 實時數據儀表板 (Dashboard)
人物資料：即時顯示 LV、轉生次數、HP/MP、攻防敏、四屬性（地水火風）、以及詳細的三圍數值（體力/腕力/耐力/速度）。

寵物監控：
支援顯示 5 隻攜帶寵物的詳細數值。
顯示 成長數據（LV, EXP, 欠缺經驗）。
狀態顯示：整合顯示寵物當前狀態（戰鬥、騎乘、郵件、等待、休息），並以對應顏色區分。
動態屬性：僅顯示數值大於 0 的屬性，保持介面簡潔。

3. 遊戲功能修改 (Memory Patching)
透過修改記憶體特徵碼 (AOB) 實現以下功能：
遊戲加速：調整遊戲邏輯速度。
快速行走：移除移動延遲。
穿牆行走 (No Clip)：允許角色穿越障礙物。
隱藏視窗：利用 WinAPI 隱藏遊戲視窗以節省資源或掛機。

4. 高度最佳化的 UI 體驗
DPI 感知與解析度適配：
內建強大的佈局引擎，自動偵測 Windows DPI 縮放 (100%, 125%, 150%)。
根據螢幕解析度（以 4K 為基準）動態計算所有 UI 元素的大小、間距與字體。
支援手動微調不同 DPI 下的版面參數。

效能優化：
採用 精確更新 (Granular Update) 機制，僅重繪數值發生變動的 UI 項目，極大降低 CPU 佔用並消除介面卡頓。
記憶體讀取與 UI 渲染分離（Producer-Consumer 模式）。

專案結構 (File Structure)檔案名稱用途描述
main.py	程式入口點。負責視窗管理、執行緒協調、DPI 偵測邏輯、以及接收 Worker 傳回的數據並更新 UI。
app_ui.py	UI 佈局核心。定義所有 Tkinter 元件、Canvas 繪圖邏輯，以及負責處理複雜的 DPI/解析度縮放參數計算。
memory_worker.py	後台執行緒。使用 pymem 讀取遊戲記憶體，解析指針結構 (Pointer Chains)，並將數據打包傳送給主執行緒。
constants.py	常數定義。包含記憶體偏移量 (Offsets)、AOB 特徵碼、顏色定義表、以及最大客戶端數量設定。
ui_components.py	UI 元件庫。包含自定義的元件，如 ScrollableFrame (可捲動框架)。
utils.py	工具函式。包含權限檢查 (is_admin)、Big5 字串解碼、數值格式化等通用功能。

注意事項 (Notes)
本程式依賴特定的記憶體位址與特徵碼 (constants.py)，若遊戲版本更新，可能需要更新相關偏移量 (Offsets)。
程式啟動時會自動偵測系統 DPI 與解析度，若版面顯示異常，可於 app_ui.py 頂部的參數區進行手動調整。
